Web Developer Kurs 2022/23

In diesem Kurs lernst du die Grundlagen der Webentwicklung

Kursinhalte

Aufbauend auf HTML und CSS-Kenntnissen steigend ie Teilnehmer in die faszinierende Welt der Webentwicklung ein. In dieser Berufsbegleitenden Ausbildung werden Web-Standards und State-of-the-Art Technologien anhand von praxisorientierten Beispielen vermittelt.

Die Lerninhalte erstecken sich von der Architektur von verteilen Softwaresystemen, deren Anforderungsmanagement ueber die Geschichte und Grundlagen der Webentwicklung, bis hin zur Programmierung komplexer Loesungen nach neuesten Technologiestandards. Durch die neu erlernten Skills erfuellen sie optimal die Anforderungen fuer den staendig erhoehten Bedarf an Web Developer. Der Kurs ist fuer Quereinsteiger geeignet.

Themen

  • Git
  • HTML & CSS
  • Javascript
  • Typescript
  • Moderne Frontend Frameworks und Libraries - React
  • Abschlussprojekt

Wir lernen in dem Kurs auch, wie wir Technologie-Dokumentationen verwenden. Da gibt es zum Beispiel die MDN web docs.
